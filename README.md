# Proyecto VCS

Proyecto de práctica para el manejo de sistemas de control de versiones con GitHub.

## Instalación

1. Clonar el repositorio:
git clone https://github.com/TU_USUARIO/proyecto-vcs.git

2. Entrar a la carpeta:
cd proyecto-vcs

## Uso

Ejecutar el archivo de calculadora con Node.js:
node calculadora.js

## Ejemplos

- suma(5, 3) → 8
- resta(10, 4) → 6
- multiplicacion(3, 4) → 12
